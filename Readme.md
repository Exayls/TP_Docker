### TP Docker

le TP comporte 3 scripts qui permettent exécuter les actions demandées.

##### étape 1

Le script "loadBalancing.sh" permet de lancer 4 docker nginx avec un proxy. Quand on le lance sur navigateur, les docker nginx sont alternés.



##### étape 2

le script "docker_build_opencv.sh" permet de build le docker  et le script "docker_run_opencv.sh" de le lancer.
